# 弈彩 YiCai — 盈利模式方案

> 现实结论：**广告（AdSense）是最现实的主力，但有两个前提：①你有流量；②内容能过 AdSense 的彩票政策。**
> 站点已就绪——填入你的 AdSense 发布商 ID 即可开始；其余变现路子作为补充。

最后更新：2026-06-20

---

## 1. 一句话现实

- 免费的"开奖数据 / 工具"站，主力变现就是**展示广告**（Google AdSense / 联盟广告）。
- 但：**变现是流量的下游**。现在站点自然流量还很小（平均排名第 10、日点击个位数），先把 SEO 做上去，广告才有意义。
- 彩票/赌博类内容，AdSense 审核严，**可能被"限制广告"甚至拒登**——所以广告优先放在"娱乐工具 / 纯数据统计"页，而不是六合彩这类敏感页。

## 2. 已经为你准备好的（本次提交）

| 文件 | 作用 |
|------|------|
| `frontend/public/ads.txt` | AdSense 授权文件，填入 pub-ID 即生效 |
| `frontend/src/components/AdSlot.vue` | 广告位组件：**没填 ID 时不渲染、不影响页面**；填了 ID 自动加载广告 |
| `/generate` 页底部 | 已放一个 `<AdSlot>`（当前不显示，配置后自动出广告） |
| `index.html` | 已有 GA4 + Consent Mode v2（"AdSense 友好"，符合欧盟同意要求） |

## 3. 开始赚钱的步骤（需要你做的部分）

1. **申请 Google AdSense**：用站点 `https://yicai.ckl.hk` 申请（[adsense.google.com](https://adsense.google.com)）。需要站点有一定内容和流量，审核通常几天到几周。
2. 拿到**发布商 ID**（形如 `ca-pub-1234567890123456`）后：
   - 把它填进 `frontend/src/components/AdSlot.vue` 顶部的 `ADSENSE_PUBLISHER_ID`；
   - 把 `frontend/public/ads.txt` 里那行注释去掉、替换 ID。
3. 在 AdSense 后台为每个广告位生成 `data-ad-slot` 数值，填到对应的 `<AdSlot slot="...">`。
4. 提交 → 构建 → 上线（auto-deploy 会自动重建）。
5. （把 ID 发我，我可以直接帮你接上并多放几个合规广告位。）

## 4. 广告放哪 / 不放哪（合规）

✅ **适合放广告**（娱乐工具 + 纯数据统计，政策风险低）：
- `/generate` 模拟选号、`/ssq/*`、`/qxc/*`、`/lottery-data`、`/lottery-frequency`、走势/频率等数据页。

⚠️ **谨慎 / 少放**（六合彩相关，AdSense 对赌博内容更敏感）：
- `/marksix/*`。可先不放，或等账号稳定后小范围测试。

📐 **放置原则**：内容上方/中间穿插 1-2 个，别堆满；移动端注意不挡操作；保留已配置的 Cookie 同意（Consent Mode）。

## 5. AdSense 彩票政策风险（务必知道）

- AdSense 对"赌博和游戏"类内容有专门政策，部分地区/内容会**限制广告**或要求**赌博内容认证**（需要相关牌照）。
- 你的"数据 / 娱乐参考"定位有帮助，但**不保证**六合彩页面能正常变现。
- 安全做法：先用娱乐工具页和数据页跑广告，观察账号状态，再决定是否扩展。

## 6. 除广告外的补充路子（都不轻松，但可叠加）

| 方式 | 说明 | 难度 |
|------|------|------|
| 会员 / 订阅 | 去广告、高级统计、数据导出、提醒推送 | 彩票数据付费意愿低，小规模可行 |
| 数据 / API 授权 | 把整理好的开奖数据/接口卖给其他站点或开发者 | 需要稳定数据源和文档 |
| 邮件沉淀 | 已有 NewsletterSignup，先攒用户再变现 | 慢，但资产属于自己 |
| 品牌 / 赞助 | 流量起来后接相关品牌 | 依赖流量规模 |

## 7. ❌ 绝对不要碰

- **投注导流 / 赌博联盟 / 售彩分成**：违背"不参与投注经营"的合规定位，**法律风险极高**（尤其面向大陆用户），也会让 AdSense 直接封号。这条路堵死。

## 8. 现实预期

- 彩票数据流量 **CPM 不高**；真正有收入的前提是**自然流量上规模**。
- 节奏：先 SEO（1-3 个月看长尾起量）→ 流量起来 → 申请 AdSense → 放广告 → 再考虑订阅/API。
- 把"模拟选号"做成留存 + 长尾入口（本次已优化），是把免费流量做大的关键一环。
